# 그사이에 — iOS

> 한 요리 안에서 동시에 돌아가는 작업들.
> **면 삶는 그사이에 소스도 함께 끝낸다** — 순서(먼저/나중)가 아니라 **병렬**을 시간축에 보여주는 앱.

파스타를 만들 때 진짜 필요한 건 "지금 이거 하고, 다음에 저거 하세요"가 아닙니다.
**면을 삶는 그사이에** 마늘 오일을 볶기 시작해서, 면이 다 될 때 소스도 같이 끝나야 합니다.
이 앱은 한 요리를 **여러 작업 레인(면 · 소스 · 마무리)**으로 쪼개, 각 작업이 언제 시작해 언제 끝나는지 — 그리고 **어디서 겹치는지(=동시에 해야 하는 것)**를 하나의 타임라인에 그립니다.

## 실행 방법

1. macOS에서 **Xcode 15 이상** 설치 (iOS 17 SDK)
2. `DongsiNaembi.xcodeproj` 더블클릭으로 열기
3. 프로젝트 설정 → **Signing & Capabilities** → 본인 Apple ID 팀 선택
4. 시뮬레이터 또는 실기기 선택 후 **⌘R** 실행

> 실기기 알림(작업 시작/완료 알림)을 받으려면 첫 실행 시 알림 권한을 허용하세요.

## 핵심 개념: 손은 하나 — "내 손" ‖ "자동"

사람은 **손이 하나**라 hands-on 작업(썰기·볶기·유화)은 절대 겹칠 수 없습니다.
불 위에서 알아서 되는 것(물 끓이기·면 삶기·졸이기)만 배경으로 돌아가죠.
그래서 레인을 두 종류로 나눕니다:

- **내 손** — hands 작업을 순서대로. **서로 겹치지 않게** 배치 (`attention` = active·instant)
- **자동** — 걸어두면 되는 것. 배경으로 겹쳐 돌아감 (`attention` = passive)

앱이 하는 일은 자동이 도는 **그 사이사이에** 손 작업(마늘 다지기 → 면 넣기 →
오일 데우기 → 볶기 → 유화)을 끼워 넣어 동선을 짜주는 것입니다.

**알리오 올리오** 예시 (리서치 기반):

| 레인 | 작업 순서 |
|------|-----------|
| 자동 | 물 끓이기(0–5:00) · 면 삶기(5:00–13:00) |
| 내 손 | 마늘 편썰기(1:00) → 페퍼론치노 다지기 → 면 넣기(5:00) → 팬 데우기 → **마늘 볶기(8:00–12:40)** → 면수 남기고 건지기 → 유화(13:00–15:00) |

→ 물·면이 알아서 익는 사이에 손 작업이 하나씩 끼워지고, **손 작업끼리는 절대 겹치지 않습니다.**
모델에는 `handsConflicts` 검증이 있어 두 손 작업이 겹치면 잡아냅니다.

## 화면 구성

| 화면 | 내용 |
|------|------|
| **홈 (RecipeListView)** | 레시피 카드. 각 카드에 미니 간트로 병렬 구조 미리보기 + "동시작업 N분" 표시. 탭하면 시작 |
| **조리 중 (CookView)** | ⭐ 간트 타임라인 위를 '지금' 선이 지나감. 상단 '지금 내 손은' 카드에 손 작업 + "걸어둔 것"(자동으로 익는 중)을 분리 표시. 진행 바를 드래그하면 그 시점으로 이동하고 타이머가 동기화. 하단 버튼은 '단계 종료'(다음 경계로 진행), 마지막엔 '완성 처리' |

디테일:
- 작업의 **시작·완료 순간마다** 로컬 알림 ("이제 마늘 볶기", "면 완료") — 백그라운드에서도 동작
- 주의도 아이콘: 🔥 지켜보기 / ⏳ 그사이에 방치 OK / 👆 한 번 동작
- 조리 중 화면 항상 켜짐 (젖은 손 UX), 완성 시 햅틱 + 사운드

## 파일 구성

```
DongsiNaembi/
├── DongsiNaembiApp.swift    # 앱 진입점
├── Theme.swift              # 크림·테라코타 팔레트, 시간 포맷터
├── Models.swift             # Recipe / RecipeStep / Attention / StepStatus
├── Recipes.swift            # 레시피 DB (알리오 올리오, 토마토 크림 파스타)
├── CookSession.swift        # 병렬 조리 엔진 + 작업별 시작/완료 알림
├── RecipeListView.swift     # 홈: 레시피 고르기
└── CookView.swift           # 조리 중: 병렬 간트 타임라인
```
